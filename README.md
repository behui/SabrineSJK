# Images du site

## Image hero (hero-bg.jpg)

Pour télécharger l'image de fond du hero :

1. Ouvrez cette URL dans votre navigateur : https://images.unsplash.com/photo-1506126613408-eca07ce68773?w=1920&q=80
2. Faites un clic droit sur l'image → "Enregistrer l'image sous..."
3. Enregistrez-la sous le nom `hero-bg.jpg` dans ce dossier `images/`
4. Commitez et poussez sur GitHub

## Format recommandé

- Format : JPEG
- Résolution recommandée : 1920px de largeur minimum
- Poids : optimiser à < 500KB si possible

## Autres images

Pour ajouter d'autres images au site :
- Placez-les dans ce dossier `images/`
- Référencez-les dans le HTML avec : `src="images/nom-image.jpg"`
